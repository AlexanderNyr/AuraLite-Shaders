> 🌐 **Языки:** [English](README.md) · **Русский**

**AuraLite** — это современный, лёгкий и максимально оптимизированный шейдпак, построенный на стандарте **OpenGL 4.6 / GLSL 460**. Он специально разработан и **протестирован для Minecraft 1.16.5 – 26.1.2 с Sodium + Iris** (а также совместим с **OptiFine**).

AuraLite дарит захватывающий, реалистичный визуальный опыт, не перегружая экран раздутыми эффектами пост-обработки (вроде агрессивного motion blur, тяжёлого bloom или screen-space отражений), обеспечивая **максимальный FPS и стабильное время кадра** на современных GPU.

## 🆕 Что нового в v0.2.7 — *Реализм, TAA и совместимость*

Версия **0.2.7** — практическое обновление реализма, стабильности и совместимости поверх v0.2.6. Добавлены новые эффекты для старших пресетов, исправлена радуга после дождя, расширены языки и собран корректный shaderpack ZIP с корневой папкой `shaders/`.

### 🌈 Исправления неба и погоды
* **Надёжная радуга после дождя** — рендер радуги и moonbow перенесён в `gbuffers_skybasic`, чтобы эффект стабильно появлялся на Iris/OptiFine, где sky-ветка composite может не выполняться.
* **Безопаснее рендер неба** — исправлена обработка stale G-buffer/emissive данных для sky-пикселей и защита от некорректной нормализации normal.

### ☀️ Физически обоснованные лучи и тени облаков
* **Объёмные солнечные лучи (godrays)** — single scattering с затуханием Бугера—Ламберта, фазой Хеньи—Гринстейна / Ми, высотной плотностью аэрозолей, shadow-map окклюзией, погодным ослаблением и учётом облачных теней.
* **Прозрачные процедурные тени от облаков** — облака отбрасывают мягкие тени разной прозрачности: перистые почти незаметные, средние умеренные, кучевые/грозовые более плотные.

### 🎞️ TAA и баланс пресетов
* **Temporal Anti-Aliasing (TAA)** — темпоральный resolve с репроекцией движения, историей предыдущего кадра, neighborhood clipping и защитой от ghosting.
* **Новый профиль VERY_LOW** — настоящий минимальный пресет для максимального FPS.
* **Перебалансировка профилей** — тени облаков начинаются с MED, godrays и TAA с HIGH, SSAO с ULTRA, а EXTREME использует самые тяжёлые варианты.

### 🌐 Локализация и упаковка
* **Локализация расширена до 59 языковых файлов** — добавлены региональные языки и совместимые алиасы, включая `fil_ph.lang` и legacy `tl_ph.lang` для Filipino/Tagalog.
* **Правильная структура ZIP** — архив shaderpack теперь содержит корневую папку `shaders/`, как ожидает Minecraft.

## 🆕 Напоминание — что появилось в v0.2.5 — *Settings Fix & Cloud/Lighting Refinement*

Версия **0.2.5** — обновление полировки и стабильности: исправлены давние баги рендера, добавлен новый профиль качества **EXTREME** с экранным затемнением (SSAO), полностью переработана система облаков для стабильного рендера при повороте камеры, улучшены туман, полярное сияние и освещение воды. Основной пост-проход (`composite.fsh`) вырос до **~1 422 строк** GLSL, сохранив прежнюю лёгкую философию и целевые показатели FPS.

### ☁️ Переработка системы облаков
* **Камерно-стабильная выборка** — облака больше не меняют вид и не «плывут» при повороте камеры. Сэмплы теперь привязаны к высоте через мировые координаты вместо экранного джиттера.
* **Независимый ветровой сдвиг по слоям** — слои Cirrus, Altocumulus, Altostratus и Cumulus перемещаются в собственных ротированных/сдвинутых доменах с разными ветровыми смещениями — верхние слои больше не копируют нижние.
* **Ветер влияет на облака** — настройка `WIND_SPEED` (Лёгкий / Бриз / Шквал) теперь управляет и дрейфом облаков; штормы ускоряют движение слоёв.
* **Дальность прорисовки облаков** (`CLOUD_DISTANCE`) — новая 4-ступенчатая настройка (*Близкая / Стандартная / Дальняя / Очень дальняя*), автоматически масштабирующаяся по профилям. Диапазон расширен: LOW 3 000 м → ULTRA 16 000 м для облачных палуб до горизонта.
* **Эффект нахождения внутри облака** — при полёте внутри облачного слоя камеру окутывает мягкая бело-серая объёмная дымка.
* **Более мягкое самозатенение** — низ облаков больше не проваливается в чёрный; добавлен амбиентный подъём для естественно тёмных, но читаемых облачных оснований.
* **Улучшенный серебристый контур / фазовое освещение** — предвычисленный фазовый член Ми обеспечивает мягкое золотое свечение по краям облаков без дополнительных шумовых вызовов.
* **Мягкие границы слоёв** — вертикальное затухание на каждом облачном слое устраняет резкие границы плотности.

### 🌟 Новый профиль EXTREME и SSAO
* **Профиль EXTREME** — пятый уровень качества, добавляющий **SSAO/SAO-контактное ambient occlusion** — экранное затемнение в углах, под блоками и на стыках геометрии.
* **Сила SSAO** — 3 ступени (*Мягкая / Сбалансированная / Глубокая*). SSAO включён на профилях **HIGH**, **ULTRA** и **EXTREME**.

### 🔧 Исправления и улучшения рендера
* **Исправление PBR-спекуляра** — добавлен недостающий множитель NdotL (косинус) для корректного BRDF по Куку-Торрансу. Блики теперь правильно тусклые при высоком солнце и яркие при скольжении.
* **Исправление slope-bias теней** — bias теперь использует необработанный NdotL (без light wrap) для предотвращения артефактов shadow acne.
* **Исправление G-буфера воды** — `gbuffers_water` теперь выводит полные G-буферные данные (`DRAWBUFFERS:012`) — lightmap, PBR и нормали — чтобы composite-проход освещал воду корректно вместо чтения устаревших данных ландшафта.
* **Унифицированный спекуляр воды** — собственный Blinn-Phong спекуляр воды удалён; GGX PBR из composite теперь обрабатывает весь водный спекуляр с корректным Френелем и моделью микрофасеток.
* **Детекция эмиссивных пикселей** — порталы и самосветящиеся поверхности теперь используют `colortex2.a < 0.5` как флаг эмиссии; composite пропускает сцинтное освещение и выводит космическую плазму как есть.
* **Исправление тайминга заката/сумерек** — `/time set 12800` остаётся красным/тёплым вместо резкого перехода в нейтральный; окно сумерек расширено до 11 000–13 000.
* **Удалён дублирующий вызов рендера** — убран второй `projectAndDivide()` в composite; viewPos вычисляется однократно.

### 🌌 Исправления полярного сияния
* **Рендер сияния перенесён в `gbuffers_skybasic`** — исправляет невидимость авроры на некоторых пайплайнах Iris/OptiFine, где composite-ветвь для неба не запускалась.
* **Исправлена маска «север»** — старый `smoothstep(0.0, -0.65, ...)` давал неопределённый результат на некоторых GPU; заменён корректной маской северного направления.
* **Реальное определение холодного биома** — `AURORA_MODE=1` (Только в холодных биомах) теперь использует **настоящий biome custom uniform** (`biome_category`, `biome_precipitation`, `temperature`) вместо ненадёжной эвристики по fogColor. Запасной вариант через fogColor сохранён для загрузчиков, не поддерживающих uniform.

### 🌫️ Улучшения приземного тумана и атмосферного тумана
* **Реалистичный атмосферный туман** — плотность тумана теперь учитывает высоту (аэрозольная концентрация), длину пути к горизонту и нахождение в помещении/на открытом воздухе через skylight.
* **Переработка приземного тумана** — слой низкого тумана (Y ≈ 60–70) теперь использует крупные медленные полотна + мелкий шум разбиения, оптическое накопление с расстоянием вместо резкого обрезания на 120 м, расширенное окно рассвета/заката и сохранение тумана на основе влажности.
* **Эффект нахождения внутри тумана** — при нахождении игрока внутри туманного слоя на весь вид накладывается тонкая пелена прямого рассеяния.

### 🛠️ Улучшения профилей и интерфейса
* **Селектор `<profile>` на главном экране** — пресеты качества (LOW / MED / HIGH / ULTRA / EXTREME) теперь видны и переключаются прямо на основном экране настроек.
* **Уведомление о переводе** — новая запись `⚠ Перевод может содержать ошибки` предупреждает, что некоторые строки локализации могут быть неточными.
* **POM отключён во ВСЕХ пресетах** (включая ULTRA) — POM нестабилен на некоторых ресурспаках / GPU-драйверах; можно включить вручную.
* **Все профили используют реалистичный терминатор** (`LIGHT_WRAP=1` — Lambert) по умолчанию.
* **Полнота profile.LOW** *(v0.2.3)* — профиль LOW теперь содержит все ключи настроек для надёжного переключения.
* **Локализация расширена** — теперь поддерживается **59 языковых файлов**, включая региональные варианты и алиасы совместимости вроде `fil_ph` и legacy `tl_ph`.

### 📋 Накопленные исправления из v0.2.3 и v0.2.4
* Параметры `CLOUD_HEIGHT` / `CLOUD_THICKNESS` теперь реально управляют геометрией облаков.
* Добавлен `#define GROUND_MIST` — переключатель в `shaders.properties` теперь работает.
* `SUN_TEMPERATURE` применяется к кривой Кельвина в `gbuffers_skybasic`, совпадая с `composite.fsh`.
* Все вызовы `texture2D()` заменены на `texture()` для совместимости с GLSL 460.
* Удалён мёртвый код: неиспользуемые `noise/fbm` функции в `gbuffers_terrain`, неработающий `renderVolumetricClouds()` в `gbuffers_skybasic`, лишний varying `mc_EntityOut`.

## ✨ Краткий обзор возможностей

### ☁️ 1. Метеорологически точные 3D-объёмные облака (с возможностью пролёта!)
AuraLite содержит полностью физическую, пролетаемую систему 3D-облаков на базе **12-шагового Raymarching'а** в мировых координатах:
* **Настоящее 3D-пространство:** Облака парят на физической высоте (настраиваемая базовая высота). Можно взлететь, войти в плотную туманную облачность и подняться над облаками, чтобы увидеть бескрайнее море пушистых кучевых облаков.
* **Эффект нахождения внутри облака** *(v0.2.5)*: При полёте внутри облачного слоя камеру окутывает мягкая объёмная бело-серая дымка.
* **Независимый ветровой сдвиг слоёв** *(v0.2.5)*: Каждый облачный слой (Cirrus, Altocumulus, Altostratus, Cumulus) перемещается в собственном ротированном домене с настройкой `WIND_SPEED`.
* **Дальность прорисовки облаков** *(v0.2.5)*: Новая 4-ступенчатая настройка (*Близкая / Стандартная / Дальняя / Очень дальняя*) — от 3 000 м до 16 000 м облачных палуб до горизонта.
* **Самозатенение по закону Бугера—Ламберта:** Реалистичное поглощение света делает низ облаков плотным и тёмным, в то время как верхушки сияют ярко-белым и золотым.
* **Рассеяние Ми (серебристая кайма):** Взгляд в сторону солнца порождает золотое сияющее гало по краям облаков.
* **Грозовая облачность:** Во время дождя (`/weather rain`) пушистые кучевые облака автоматически расширяются, темнеют и сливаются в зловещий тяжёлый штормовой фронт **Nimbostratus / Cumulonimbus**.

### 🌠 2. Живое ночное небо *(с v0.2.0)*
Ночное небо больше не статичное звёздное полотно — это полностью процедурный космос:
* **Северное сияние (Aurora Borealis):** Реалистичные, текучие сияющие ленты, переливающиеся в верхней части неба. Режимы: *Выключено / Только в холодных биомах / Всегда включено*, с независимыми настройками **скорости** и **яркости**. *(v0.2.5: рендерится в `gbuffers_skybasic` для надёжности; определение холодного биома через настоящие biome-юниформы.)*
* **Туманность Млечного Пути:** Едва заметная диагональная коричневатая галактическая полоса мягко светится над горизонтом, с регулируемой яркостью.
* **Процедурные звёзды:** Независимые ползунки **яркости** и **плотности** позволяют выбрать между несколькими крупными точками и плотным «хабблоподобным» небом. Звёзды мерцают и переливаются в реальном времени.
* **Затухающая радуга:** После окончания дождя на небе появляется мягкая радуга, плавно исчезающая вместе с уменьшением uniform-переменной `wetness`. Яркость и насыщенность настраиваются.

### ☀️ 3. Аналитическое освещение Солнца и Луны — *усилено в v0.2.2, доработано в v0.2.5*
* **Кривая чёрного тела (Tanner Helland) для солнца:** Цвет солнечного света динамически рассчитывается в реальном времени на основе угла высоты солнца по физически корректной **кривой Кельвина для абсолютно чёрного тела** (выбор: *Холодный / Реалистичный / Тёплый золотой*). Это даёт фотореалистичные цвета заката/рассвета (~1800K–2200K), тёплые «золотые часы» (~2800K) и чистый яркий полуденный свет (~5700K–5800K).
* **Атмосферное затухание по Бугеру—Ламберту:** Интенсивность солнечного света динамически падает по мере приближения солнца к горизонту из-за рассеяния в плотной атмосферной массе:
  $airMass = \frac{1}{\sin(\alpha) + 0.15 \cdot (\alpha_{deg} + 3.885)^{-1.253}}$
  Это создаёт невероятно мягкие, насыщенные и захватывающие переходы заката и рассвета!
* **Независимая яркость солнца и луны** *(v0.2.2)*: 4-ступенчатые мастер-ползунки позволяют сделать день ярче (*Палящее*) или погрузить ночи в полную темноту (*Кромешная ночь*).
* **Цветовая температура луны** *(v0.2.2)*: выбор между *Ледяным голубым*, *Серебристым* (физически точные 4100K) или *Тёплым кремовым* (урожайная луна).
* **Гало вокруг солнца (рассеяние Ми)** и **Усиленное свечение восхода/заката** *(v0.2.2)* — тёплые рассеивающие эффекты на ландшафте при низком солнце.
* 🆕 **Расширенное окно сумерек** *(v0.2.5)*: Освещение заката/рассвета остаётся тёплым/красным при `/time set 12800` вместо резкого перехода в нейтральный.
* **Чёткие круглые диски Солнца и Луны:** Кастомные процедурные, идеально круглые, сглаженные диски рисуются на небесном куполе со светящимися коронами и мягким ореолом рассеяния.

### 👥 4. Мягкие тени, атмосферные тёмные ночи и уютный свет — *усилено в v0.2.2, доработано в v0.2.5*
* **Мягкие тени с ротированным Poisson-диском** *(v0.2.2)*: заменяет старое фиксированное ядро 3×3 PCF. Три уровня качества — *Резкие / Мягкие / Очень мягкие* — дают естественно выглядящую полутень на shadow-картах до 4096×4096.
* 🆕 **Исправление slope-bias теней** *(v0.2.5)*: bias теперь использует необработанный NdotL для предотвращения артефактов shadow acne.
* **Контроль дальности теней** *(v0.2.2)*: задайте дальность рендера динамических теней — *60м / 80м / 120м / 160м* — для тонкой настройки производительности или качества.
* **Оттенок теней** *(v0.2.2)*: реалистичный холодный синий оттенок дневных теней под открытым небом (или нейтральный / тёплый — по вашему вкусу).
* **Подъём ambient'а** *(v0.2.2)*: управляйте тем, насколько тёмными выглядят затенённые области ночью и в пещерах.
* **Мягкость терминатора (Light Wrap)** *(v0.2.2)*: выберите физический Lambert, мягкий «фотографический» wrap или стилизованный вид. *(v0.2.5: все профили по умолчанию используют реалистичный Lambert.)*
* 🆕 **SSAO / SAO контактное затемнение** *(v0.2.5)*: экранное затемнение в углах и на стыках геометрии. Включено на профилях ULTRA+.
* **Глубокие тёмные ночи (в 2 раза темнее):** Ночное амбиентное освещение, интенсивность лунного света и тумана уменьшены вдвое по умолчанию, создавая невероятно атмосферные тёмные ночи. Пещеры и леса погружены в абсолютную темноту — без факелов не обойтись (попробуйте новый пресет луны *Кромешная ночь* для усиления эффекта).
* **Тёплый свет блоков:** Факелы, фонари и лава излучают уютное золотисто-янтарное свечение с физически точным квадратичным затуханием.
* 🕯️ **Уютное мерцание факелов** *(с v0.2.0)*: Реалистичные анимации мерцания пламени факелов, костров и фонарей добавляют живую тёплую атмосферу вашим убежищам. Также учитывается свет от предметов в руках (`heldBlockLightValue`).

### 🌊 5. Физический Френель-эффект воды и серебристая лунная дорожка — *доработано в v0.2.5*
* **Эффект Френеля:** Отражательная способность воды математически рассчитывается на основе угла обзора. Взгляд прямо вниз даёт кристальную прозрачность, а взгляд в сторону горизонта превращает воду в высокоотражающую глянцевую поверхность, отражающую небесный купол.
* **Серебристая лунная дорожка:** Зеркальное отражение лунного света на ряби воды усилено в **4,5 раза**. В полночь по волнам струится сверкающая серебряная лунная дорожка.
* 🆕 **Унифицированный GGX PBR спекуляр воды** *(v0.2.5)*: Старый Blinn-Phong спекуляр воды заменён GGX-моделью микрофасеток из composite с корректным Френелем — физически согласовано с PBR ландшафта.
* **3D-геометрические волны:** Вершинный шейдер физически смещает меш воды в реальном времени; волны реагируют на `rainStrength` / `thunderStrength` — во время шторма море становится более бурным.
* **Раздельный контроль ряби и бликов** *(с v0.2.0)*: `WATER_RIFFLES` (Спокойная / Стандартная / Бурная) и `WATER_SPECULAR_STRENGTH` (Мягкое / Стандартное / Сверкающее) настраиваются независимо для идеального настроения воды.
* **Никаких feedback-глюков:** Архитектура максимально стабильна, не использует чтение feedback-loop depth-буфера, что гарантирует безглючный рендер на всех GPU.

### 🌧️ 6. Динамические мокрые поверхности *(с v0.2.0)*
* **Мокрые отражения (Wet Reflections):** Во время дождя твёрдые блоки — трава, земля, камень — темнеют и становятся глянцевыми, отражая небо под открытым небом. Автоматически отключаются под крышей.
* **Низкий приземный туман (Y ≈ 60–70)** *(доработан в v0.2.5)*: Мягкие полотна тумана скользят по поверхности воды и льда на рассвете и закате. Используются крупные медленные полотна + мелкий шум разбиения с оптическим накоплением по расстоянию для естественной радиационной дымки. Влажность от дождя/wetness увеличивает время удержания тумана.
* **Эффект нахождения внутри тумана** *(v0.2.5)*: Нахождение игрока внутри слоя тумана создаёт тонкую пелену прямого рассеяния на весь вид.
* **Распознавание грозы:** Шейдеры различают обычный дождь и полноценную грозу через uniform `thunderStrength`, усиливая темноту облаков и силу волн соответственно.

### 🌿 7. Оживлённая листва
* Анимации колыхания листвы дуба/ели/берёзы, высокой травы, цветов, лиан, кувшинок и сельскохозяйственных культур.
* Мягкая анимация на аппаратно оптимизированных синусоидах и временных константах.
* 🧊 **Фикс льда** *(с v0.2.0)*: лёд / упакованный лёд / голубой лёд / замороженный лёд помечены отдельным block ID, отключающим колыхание и преломление, что устраняет давний визуальный артефакт.

### 💎 8. Полная поддержка LabPBR 1.3 + POM — *PBR доработан в v0.2.5*
* **3D-карты нормалей:** Реалтайм-матрицы **TBN (Tangent-Binormal-Normal)** создают настоящую трёхмерную глубину на блоках (трещины камня, швы кирпича), динамически реагирующую на угол освещения.
* **Зеркальные отражения (GGX Microfacet):** Полированные поверхности дают резкие глянцевые блики, а металлические поверхности (золото, медь, железо) окрашивают зеркальное отражение в цвет родного альбедо блока. *(v0.2.5: добавлен корректный множитель NdotL для точности BRDF по Куку-Торрансу.)*
* 🧱 **Parallax Occlusion Mapping (POM)** *(с v0.2.0)*: Настоящий попиксельный рельеф блоков, выступающий из поверхности. Настраиваемые `POM_DEPTH` (1–3) и `POM_STEPS` (1–4). По умолчанию отключён во всех профилях для стабильности; можно включить вручную. Рекомендуется отключать на несовместимых ресурспаках.
* *Прозрачный fallback:* Автоматически откатывается к красивым плоским ванильным текстурам, если PBR-ресурспак не активен.

### 🌀 9. Космический портал в Нижний мир *(с v0.2.0, улучшен в v0.2.5)*
Стандартная текстура портала Нижнего мира процедурно превращается в **закрученный 3D-вихрь плазмы** — анимированную фиолетово-маджентовую космическую энергию, пульсирующую гипнотической глубиной. Прописан через выделенный block ID `10006` в `block.properties`. *(v0.2.5: пиксели портала помечены как эмиссивные — composite пропускает сценическое освещение и отображает плазму как есть.)*

### 🎬 10. Кинематографичная пост-обработка
* **Несколько кривых тонмаппинга** *(с v0.2.0)*: Выбор между **Soft**, **Filmic (ACES)** или **Intense (High Contrast)** под предпочитаемое настроение.
* **Сочность цвета (Color Vibrancy)** *(с v0.2.0)*: 4-ступенчатый нелинейный контроль насыщенности (*Приглушённая / Сбалансированная / Красочная / Яркая*), благодаря которому листва сияет изумрудом, а небо выглядит сочным — без пересвета телесных тонов.
* **Экспозиция (яркость):** Приглушённая / Сбалансированная / Яркая — глобальный подъём яркости.
* **Деликатная виньетка:** Мягкое затемнение по краям экрана для усиления глубины и погружения.

### 🛡️ 11. Реалистичный атмосферный туман *(v0.2.5)*
* Плотность тумана теперь учитывает **высоту** (аэрозольная концентрация уменьшается с высотой), **длину пути к горизонту** и **нахождение в помещении / на улице** через skylight.
* Согласованный туман Бугера—Ламберта с весовым расчётом плотности по высоте — без отложенных «стен тумана».

## ⚙️ Оптимизации производительности (OpenGL 4.6 Native)

AuraLite построен с нуля для максимального FPS, используя нативные аппаратные операции OpenGL 4.6:
* **Hardware FMA (Fused Multiply-Add):** Вся математика волн и колышущейся листвы компилируется в нативные однотактные аппаратные операции (`fma`), сокращая число GPU-циклов.
* **Битовая генерация шума:** Медленные трансцендентные функции с плавающей точкой (`fract(sin(dot(...)))`) заменены на сверхбыстрые **целочисленные битовые PCG-хэши** с использованием `floatBitsToUint` и `uintBitsToFloat`.
* **Раннее завершение лучей:** Объёмный raymarching мгновенно прерывается, как только пропускание облака падает ниже 2%, экономя ресурсы рендера.
* **Никаких глюков прозрачности в руке:** Предметы в руках, частицы и мобы рендерятся в отдельном стабильном пути без оверхеда тангенциальных матриц — баг «полупрозрачной руки» устранён.
* **Удаление мёртвого кода** *(v0.2.3–v0.2.5)*: Удалены неиспользуемые функции noise/fbm, мёртвый код raymarching'а облаков из `gbuffers_skybasic` и лишние вызовы рендера для сокращения времени GPU-компиляции.
* **Масштабирование по профилям:** Все фичи (POM, сияние, SSAO, мерцающий свет, мокрые отражения, приземный туман, дальность теней, дальность облаков, гало солнца и т.д.) умно распределены по профилям **LOW / MED / HIGH / ULTRA / EXTREME**, чтобы слабые системы не платили за эффекты, которые им не по силам.

## 📥 Установка

1. Скачайте **`AuraLite-Shaders-v0.2.7.zip`** из раздела [Releases](https://github.com/AlexanderNyr/AuraLite-Shaders/releases) справа.
2. Откройте директорию Minecraft (например, `%appdata%/.minecraft` на Windows).
3. Поместите скачанный `.zip` в папку **`shaderpacks`** (**не** распаковывайте его!).
4. Запустите поддерживаемую версию Minecraft (**1.16.5 – 26.1.2**) с профилем, в котором установлены **Sodium + Iris** или **OptiFine**.
5. В игре зайдите в **Настройки → Настройки графики → Шейдеры**, выберите **AuraLite** и нажмите **Применить**.

> 💡 В репозитории лежат папки с исходниками для каждого релизного снапшота: от `shaders v0.2.0/` до `shaders v0.2.7/`. Актуальная версия — **v0.2.7**. Обычным пользователям следует скачивать упакованный ZIP из релизов; разработчики могут изучать любую из папок напрямую.

* **Скорость ветра** — `Лёгкий / Бриз / Шквал`

* **Волны на воде** — Вкл./Выкл. 3D-вершинные волны.
* **Плотность воды** — `Прозрачная / Сбалансированная / Глубокая` — настройка прозрачности.
* **Сила ряби воды (`WATER_RIFFLES`)** — `Спокойная / Стандартная / Бурная` — мелкая рябь на normal map.
* **Яркость бликов воды (`WATER_SPECULAR_STRENGTH`)** — `Мягкая / Стандартная / Сверкающая` — яркость отражений солнца/луны на ряби.

### `[Небо и облака]`
* **Объёмные 3D-облака** — Вкл./Выкл. raymarching-облака.
* **Высота облаков** — `Низкая (~110м) / Стандартная (~160м) / Высокая (~240м)`
* **Толщина облаков** — `Тонкие (Cirrus) / Стандартные (Cumulus) / Плотные (грозовые)`
* 🆕 **Дальность облаков** *(v0.2.5)* — `Близкая / Стандартная / Дальняя / Очень дальняя` — максимальная дальность прорисовки объёмных облаков.
* 🆕 **Тени от облаков** *(v0.2.7)* — прозрачные процедурные тени от плотности облаков.
* 🆕 **Сила теней от облаков** *(v0.2.7)* — `Мягкие / Сбалансированные / Драматичные`.
* 🆕 **Солнечные лучи (Godrays)** *(v0.2.7)* — физически обоснованные объёмные лучи single scattering.
* 🆕 **Качество солнечных лучей** *(v0.2.7)* — `Быстрое / Сбалансированное / Высокое`.
* **Северное сияние** — `Выключено / Только в холодных биомах / Всегда включено`
* **Скорость сияния** — `Медленная / Стандартная / Быстрая`
* **Яркость сияния** — `Мягкая / Стандартная / Сияющая`
* **Яркость Млечного Пути** — `Тусклая / Стандартная / Яркая`
* **Яркость звёзд** — `Слабая / Стандартная / Блестящая`
* **Плотность звёзд** — `Мало / Стандартная / Густо`
* **Интенсивность радуги** — `Деликатная / Сбалансированная / Яркая` — радуга после дождя.

### `[Пост-обработка и туман]`
* **Плотность тумана** — `Низкая / Средняя / Высокая` — атмосферный туман у горизонта.
* **Низкий приземный туман (`GROUND_MIST`)** — Реалистичный рассветный/вечерний радиационный туман на Y ≈ 60–70.
* **Экспозиция (яркость)** — `Приглушённая / Сбалансированная / Яркая`
* **Сочность цвета (`COLOR_SATURATION`)** — `Приглушённая / Сбалансированная / Красочная / Яркая`
* **Контраст изображения (`CONTRAST`)** — `Мягкий / Filmic (ACES) / Интенсивный (высокий контраст)` — выбор кривой тонмаппинга.
* 🆕 **Темпоральное сглаживание (`TAA`)** *(v0.2.7)* — темпоральный resolve с репроекцией движения для старших пресетов.
* 🆕 **Сила TAA** *(v0.2.7)* — `Лёгкая / Сбалансированная / Стабильная`.
* **Виньетка** — Вкл./Выкл. кинематографичное затемнение по углам.
* (Скрытое) **Мокрые отражения (`WET_REFLECTIONS`)** — Глянцевая мокрая земля во время дождя (включено по умолчанию в MED+).
